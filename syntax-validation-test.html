C语言实时语法校验功能测试

测试1: 缺少分号错误

复制以下代码到编辑器中，应该显示错误：

#include <stdio.h>

int main() {
    int x = 10  // 缺少分号
    printf("Hello World")  // 缺少分号
    return 0;
}
测试2: 括号不匹配错误

复制以下代码到编辑器中，应该显示错误：

#include <stdio.h>

int main() {
    if (x > 0 {  // 缺少右括号
        printf("Positive");
    }
    
    int arr[5] = {1, 2, 3, 4, 5;  // 缺少右大括号
    return 0;
}
测试3: 中文符号错误

复制以下代码到编辑器中，应该显示错误：

#include <stdio.h>

int main（） {  // 中文括号
    int x = 10；  // 中文分号
    printf（"Hello"）；
    return 0；
}
测试4: 无效include语句

复制以下代码到编辑器中，应该显示错误：

#include stdio.h  // 缺少尖括号或双引号
#include <stdio>   // 缺少.h扩展名

int main() {
    printf("Hello World\n");
    return 0;
}
测试5: 缺少main函数警告

复制以下代码到编辑器中，应该显示警告：

#include <stdio.h>

void helper() {
    printf("This is a helper function\n");
}

// 没有main函数
测试6: 正确的代码

复制以下代码到编辑器中，应该没有错误：

#include <stdio.h>
#include <stdlib.h>

int main() {
    int numbers[5] = {1, 2, 3, 4, 5};
    
    for (int i = 0; i < 5; i++) {
        printf("Number %d: %d\n", i + 1, numbers[i]);
    }
    
    if (numbers[0] > 0) {
        printf("First number is positive\n");
    } else {
        printf("First number is not positive\n");
    }
    
    return 0;
}
如何使用语法校验功能：
  1. 将上述测试代码复制到Monaco编辑器中
  2. 观察编辑器左侧的红色波浪线和错误标记

  5. 修正错误后观察标记是否消失
